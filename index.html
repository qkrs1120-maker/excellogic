격자 패턴 검증 — 실행법 및 사용 방법

▶ 실행 방법

  1. 상단 리본 데이터 탭을 클릭합니다.
  2. 분석 수준 선택 드롭다운에서 격자 패턴 검증 그룹의 항목을 선택합니다.
    · 기초 패턴 검토 (5×5) — 입문 단계
    · 심화 패턴 분석 (10×10) — 중간 단계
    · 전문 패턴 감사 (15×15) — 고급 단계
  3. 항목 불러오기 버튼을 클릭하면 목록 창이 뜹니다.
  4. 원하는 케이스를 클릭하면 시트 안에 격자 패턴이 로드됩니다.

▶ 게임 규칙 (네모로직)

네모로직은 숫자 힌트를 보고 격자 칸을 채우는 논리 퍼즐입니다.

  • 각 행·열 바깥에 숫자 힌트가 표시됩니다.
  • 힌트 숫자는 해당 줄에서 연속으로 채워진 칸의 개수를 나타냅니다.
  • 숫자가 여러 개면 각 블록 사이에 빈 칸이 최소 1칸 이상 있어야 합니다.
  • 좌클릭 또는 좌클릭 드래그: 칸을 검정으로 채웁니다 (정답 칸).
  • 우클릭 또는 우클릭 드래그: 칸을 회색으로 표시합니다 (빈 칸 메모).
  • 모든 행·열 힌트가 맞으면 검증 완료 팝업이 표시됩니다.
  • 검증 실행 버튼으로 수동 정답 확인도 가능합니다.
  • 초기화 버튼으로 현재 진행을 초기화할 수 있습니다.

수치 분석 검증 — 실행법 및 사용 방법

▶ 실행 방법

  1. 상단 리본 데이터 탭을 클릭합니다.
  2. 분석 수준 선택 드롭다운에서 수치 분석 그룹의 항목을 선택합니다.
    · 수치 입력 검토 (9×9) — 기초
    · 표준 수식 분석 (9×9) — 중급
    · 심층 통계 감사 (9×9) — 고급
  3. 항목 불러오기 버튼을 클릭하면 목록 창이 뜹니다.
  4. 케이스를 선택하면 B열 2행부터 9×9 수치 검증 표가 로드됩니다.

▶ 게임 규칙 (스도쿠)

스도쿠는 9×9 칸에 1~9 숫자를 논리적으로 채우는 수치 검증 퍼즐입니다.

  • 모든 행(가로줄)에 1~9가 한 번씩 들어가야 합니다.
  • 모든 열(세로줄)에 1~9가 한 번씩 들어가야 합니다.
  • 굵은 선으로 구분된 3×3 박스 9개에도 1~9가 각각 한 번씩 들어가야 합니다.
  • 회색 배경 칸은 고정 수치로 변경 불가입니다.
  • 흰색 배경 빈 칸을 클릭하여 선택한 후 숫자키(1~9)를 눌러 입력합니다.
  • Delete / Backspace / 0 키로 입력을 지울 수 있습니다.
  • 방향키(↑↓←→)로 칸 이동이 가능합니다.
  • 모두 채운 후 검증 실행 버튼을 눌러 정답을 확인합니다.

개인정보처리방침 및 이용약관

■ 개인정보처리방침

본 서비스(이하 "서비스")는 이용자의 개인정보를 수집하지 않습니다.

  • 서버에 데이터를 전송하지 않으며 모든 진행 상황은 브라우저 로컬 저장소(localStorage)에만 저장됩니다.
  • 광고 서비스(Google AdSense)는 제3자 쿠키를 사용할 수 있으며, Google의 개인정보처리방침이 적용됩니다.
  • 서비스 이용 시 수집되는 정보: 없음 (광고 시스템 제외)
  • 문의: 서비스 운영자에게 연락하시기 바랍니다.

■ 이용약관

제1조 (목적)
본 약관은 서비스가 제공하는 웹 기반 퍼즐 서비스의 이용 조건 및 절차에 관한 사항을 규정합니다.

제2조 (서비스 내용)
본 서비스는 격자 패턴 검증(네모로직) 및 수치 분석 검증(스도쿠) 형태의 퍼즐 콘텐츠를 제공합니다.

제3조 (금지 행위)
이용자는 다음 행위를 해서는 안 됩니다.
· 서비스의 소스코드 무단 복제 및 배포
· 서비스를 이용한 영리 목적 행위
· 광고를 차단하거나 우회하는 행위

제4조 (면책 조항)
서비스는 콘텐츠의 정확성 및 서비스의 지속성을 보장하지 않으며, 이용으로 인한 손해에 대해 책임을 지지 않습니다.

제5조 (약관 변경)
본 약관은 운영자 판단에 따라 변경될 수 있으며, 변경 시 서비스 내에 공지합니다.
