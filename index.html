🧩 네모로직 하는 법

게임 규칙: 행·열 힌트 숫자만큼 연속으로 칸을 채워 패턴을 완성하세요.

조작법:
· 좌클릭 / 드래그 → 검정으로 채우기 (정답 칸)
· 우클릭 / 드래그 → 회색으로 마크 (빈 칸 표시)
· 검증 실행 버튼 → 정답 확인

실행법: 데이터 탭 → 분석 수준 선택 → 항목 불러오기 → 항목 클릭

🔢 스도쿠 하는 법

게임 규칙: 9×9 칸에 1~9를 한 번씩만 사용하여 행·열·3×3 박스를 채우세요.

조작법: 빈 칸 클릭 후 숫자(1~9) 키보드 입력

실행법: 상단 스도쿠 탭 → 난이도 선택 → 새 문제
